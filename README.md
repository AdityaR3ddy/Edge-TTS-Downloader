# Edge TTS Downloader

A Desktop application built with Python and PyQt6 that leverages Microsoft Edge's neural TTS (Text-to-Speech) to generate and download high-quality, natural-sounding speech.

## ‚ú® Features

- **Neural Voices:** Access to the full library of Microsoft Edge neural voices across various regions.  
- **Precise Control:** Adjust Speed and Pitch in real-time.  
- **Playback Engine:** Can seek the audio once generated.  
- **Local Caching:** Efficiently handles temporary files across Windows, Linux, and macOS.  
- **One-Click Export:** Download your generated speech as high-quality MP3 files.
